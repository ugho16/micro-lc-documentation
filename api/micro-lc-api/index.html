micro-lc API

micro-lc injects an API into each element composing the layout and as property of all applications rendered as content (you can read a detailed explanation in the dedicated section).

The API can be logically clustered into the following sections.

📄️ Application state

Consume content state

📄️ Reactive communication

Reactive state object manipulation

📄️ Routing

Routing facilities

📄️ Extensions

Built-in and custom API extensions
